ΓΙΑ COMPILE ΠΟΛΛΑΠΛΩΝ ΑΡΧΕΙΩΝ JAVA ΜΕ packages

1- Make a folder, πχ myFile

2- Inside it create another folder with the name of the package in your java project

3- Inside package folder drop all .java files

4- open cmd, cd to the specific dir with the package
	πχ. εαν ο φάκελος που περιέχει το package λέγεται myFile
	τότε cd myFile
5- javac ./myFile/Ονομα_αρχείου_με_main.java

6- για εκτέλεση εφόσον είμαστε στο folder myFile τοτε: java onoma_package.onoma_main

7-εάν όλα πήγαν καλά τότε θα πρέπει να τρέχει κανονικα ο κώδικάς μας..........
